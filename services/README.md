# Services Directory

All microservices are organized in this directory.

## Structure

- `user-service/` - User management service
- `listing-service/` - Flights, Hotels, Cars listings
- `booking-service/` - Booking management
- `billing-service/` - Payment processing
- `review-logging-service/` - Reviews and logging
- `admin-analytics-service/` - Analytics and reporting
- `api-gateway/` - API Gateway for routing
- `agentic-recommendation-service/` - FastAPI recommendation service
